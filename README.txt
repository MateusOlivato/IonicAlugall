*A aplicação foi enviada sem a pasta node_modules, esta pode ser obtida quando o aplicativo for executado.*

Para o aplicativo funcionar como na apresentação do TCC, serão necesárias algumas ações.
1. Ter instalado o aplicativo "Xampp", fazendo com que este esteja com o MySQL e o Apache iniciados.
2. Existe uma pasta em meio as outras do aplicativo denominada "alugall", tal pasta deve ser copiada e colada para a pasta "htdocs" dentro dos arquivos do Xampp, o caminho padrão é "C:\xampp\htdocs".
3. Pelo MySQL Workbench, rodar o script de banco de dados que tambem fora enviado em anexo.

Cumpridos estes passos, pode-se abrir a aplicação no programa no Visual Studio Code e executar o comando "ionic cordova run browser" para que o aplicativo seja "rodado".
Basta aguardar para que o programa faça reparos ou adaptações necessárias para a execução do projeto.
